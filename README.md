# Entertainment Center Website

A multi-page website designed for an entertainment and gaming center. The project provides information about available games, upcoming events, pricing plans and registration options for visitors.

## Features

* Multi-page website architecture
* Home page with project overview
* Games catalog
* Events page
* Pricing information
* User registration page
* Contact page with inquiry form
* Responsive navigation
* Consistent styling across all pages

## Technologies Used

* HTML5
* CSS3

## Project Structure

├── index.html
├── games.html
├── events.html
├── prices.html
├── register.html
├── contact.html
└── styles.css
